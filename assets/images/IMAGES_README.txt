==================================================
  BENDER EXPORTS — IMAGE FOLDER GUIDE
==================================================

Place your images in this folder with EXACTLY
these filenames so the website works automatically:

REQUIRED FILES:
───────────────────────────────────────────────
  logo.png              → Your company logo (transparent background recommended)
                          Recommended size: 300px wide

  hero-1.jpg            → Hero slideshow image 1 (Musaza station)
  hero-2.jpg            → Hero slideshow image 2 (Nyungwe station)
  hero-3.jpg            → Hero slideshow image 3 (Nyarubaka station)
  hero-4.jpg            → Hero slideshow image 4 (Team / About)
                          Recommended size: 1920 x 1080px minimum

  about.jpg             → About section main photo (team or farm)
                          Recommended size: 900 x 1100px (portrait)

  station-musaza.jpg    → Musaza washing station photo
  station-nyungwe.jpg   → Nyungwe washing station photo
  station-nyarubaka.jpg → Nyarubaka washing station photo
                          Recommended size: 1200 x 800px (landscape)

  photostrip-1.jpg      → Photo strip panel 1 (wide, landscape)
  photostrip-2.jpg      → Photo strip panel 2 (square)
  photostrip-3.jpg      → Photo strip panel 3 (square)
                          Recommended size: 800 x 600px

  process-1.jpg         → Process section image 1
  process-2.jpg         → Process section image 2
  process-3.jpg         → Process section image 3
                          Recommended size: 800 x 500px

  impact-main.jpg       → Impact section main large image
  impact-accent.jpg     → Impact section small overlapping image
                          Recommended size: 800 x 600px

  og-image.jpg          → Social media share image (Open Graph)
                          Recommended size: 1200 x 630px

TIPS:
───────────────────────────────────────────────
  - Use JPG for photos (smaller file size)
  - Use PNG for logo (transparent background)
  - Compress images at tinypng.com before uploading
  - All images should be under 500KB each for fast loading
